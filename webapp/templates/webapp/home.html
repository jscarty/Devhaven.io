What Devhaven.io is

Devhaven.io is a newly launched programming discussion community. I believe in a clean, minimalist, and beautiful site that delivers a pure programming discussion experience, where there are no distractions of reputation and invites.The community is designed to be as friendly as possible, which is why we do not implement a reputation voting system. However, in order to make the community more connected, I'm currently pursuing an aggregation of the profile features you would like on this site. These can include an avatar picture, workplace, website, school, etc. If you have a list of these subbestions suggestions, contact Henry at henry.david.zhu@gmail.com.

I know that the site looks pretty barren at the moment, but it's the first version, and everything has to be working first.
Updates

4/20/2016: Devhaven.io is up! It's currently being hosted on a single Heroku dyno, a lightweight Linux container. We'll be planning to expand this soon, as it will only provide us 5MB of data storage.
